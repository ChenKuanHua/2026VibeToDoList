# 2026VibeToDoList (Let's Stand & Task Pomo)

  <img src="./logo.svg" alt="Logo" width="150" />
</div>

一款結合了「番茄鐘 (Pomodoro)」與「代辦事項 (To-Do List)」的極簡風辦公工具包。專為長時間久坐的辦公人士設計，強調簡約的清新美學，並且在番茄鐘休息時會隨機推送符合 **NASM CPT** 標準的伸展提示。

## 🌟 核心特色
- **極簡溫暖沙色 (Warm Sand) 介面**：低飽和色彩與平滑的微動畫，帶來無壓力的操作體驗。
- **神準的「番茄鐘 x 代辦」計時連動**：當您在追蹤特定任務的執行時長時，一旦番茄鐘進入休息時間，任務的計時器會被「自動暫停」，讓任務的數據嚴格排除上廁所與休息時間！
- **客製化預設時間**：隨時依據需求儲存專屬的工作/休息時間長度。
- **詳細的任務分類與狀態追蹤**：支援設定「公司/個人」屬性，且擁有獨立的「完成紀錄」頁面與多重篩選過濾器。
- **NASM CPT 伸展提示**：休息時隨機抽取如「坐姿梨狀肌伸展」、「胸大肌伸展」等有效放鬆動作。

## 🚀 技術架構與開發過程

<div align="center">
  <img src="./projectFlow.svg" alt="專案架構流程圖" width="800" />
</div>

本專案採用 **React + Vanilla CSS** 的單一架構 (Standalone HTML) 進行快速構建。
不需要複雜的 Node.js 環境設定，所有 React 代碼、樣式及圖示都巧妙地封裝在了單一個 `index.html` 當中，可直接透過瀏覽器開啟或部署至任何靜態伺服器！

### 開發疊代過程摘要
1. **初期規劃**：確立了極簡清新風格與溫暖沙色的主題，並制定 State 連動邏輯（Task 與 Pomodoro 時間交錯判定）。
2. **UI 元件建置**：實作進度條環型動畫、表單模態框、與平滑過渡的 Hover CSS 樣式。
3. **Standalone 轉換**：為解決在同步環境下 `npm install` 鎖檔問題，採用無伺服器的 CDN + Babel Standalone 編譯技術，使整個專案變得可以做到真正的「隨開即用」。
4. **進階優化**：因應使用回饋，追加了時間客製化存檔 (Local Storage)、雙視圖分頁系統 (目前任務 / 歷史紀錄) 以及複合式過濾器標籤機制。

## 🛠️ 如何執行
因為我們將它打包為完全靜態的檔案，您不需要安裝任何套件：
- 方式一：使用 VS Code 的 **Live Server (Go Live)** 擴充套件，對著 `index.html` 右鍵點擊開啟。
- 方式二：直接將本儲存庫部署至 GitHub Pages，即可作為雲端網頁應用程式使用！
